# 03 AIエージェント 最小構成(所要時間: 約20分)

## このステップのゴール
- 「LLMがツールを選んで呼ぶ」というエージェントの基本動作を体験する
- `@tool` で自作関数をツール化できる
- `create_agent` が返すメッセージ履歴から、どのツールが選ばれて呼ばれたかを観察できる

## エージェントとは

```
ユーザー質問 ──→ LLM ──┬─→ 「計算ツールを呼ぼう」と判断
                       │
                       ├─→ Tool 実行(Python関数)
                       │
                       └─→ ツール結果を踏まえて最終回答
```

LLM自身が「次に何をすべきか」「どのツールを使うか」を決めます。
これが可能なのは **Tool Calling 対応モデル**(Claude 3系、Amazon Nova系など)を使っているからです。

## やること

[agent.ipynb](agent.ipynb) を実行。

## Done条件

`result["messages"]` の履歴で Agent が Tool を呼んでいる様子(Ai の tool_calls → Tool メッセージ)が見え、最終回答が得られる。
