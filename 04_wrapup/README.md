## ここまでで作れるもの

- 単発のチャットアプリ
- 簡易な社内ドキュメント検索 + 回答ボット
- ツールを呼べるアシスタント(時刻取得・計算・API呼び出しなど)

これだけでも「LLMを業務で試す」レベルの PoC は十分作れます。
